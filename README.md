This repository contains the implementation of Unit Tests for the Geometry and Automotive projects. The goal was to ensure code reliability and verify all public methods using modern testing tools

Technologies Used
Language: C# Test Framework: xUnit Assertions: FluentAssertions (for better readability)

Key Features Tested
Logic: Refueling limits, fuel consumption, and odometer resets. Geometry: Coordinate movement, distance calculations, and polygonal chain length.Edge Cases: Testing invalid inputs (e.g., negative tank capacity) and empty fuel scenarios
